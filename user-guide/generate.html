Generating Reports

After completing all of the evaluation steps, it is possible to generate a report based on your findings. This report can be included in the EPUB publication or hosted on the web and linked to from the publication.

There are two output options for generating the report:

Open in new window

Choosing this option allows you to preview the final report.

You will need to save the report to your local file system using the browsers save page option. It is typically possible to find this option by right-clicking on the page.

Do no refresh or navigate away from the page before saving it. If you do, the information it contains will be lost and you will have to regenerate.

Output to text box

This option provides easier access to the HTML source. The report markup is output to a text box below the button.

You will need to copy and paste the markup to a text editor and save the file to your local file system.

If any errors are encountered while compiling the report, you will receive an alert and the errors will be listed in a message panel at the bottom of the page. Clicking on an error message will take you to the location of the problem.
