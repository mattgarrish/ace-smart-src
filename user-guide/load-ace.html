Loading an Ace Report

The most common way to begin a new evaluation of a publication is by loading the JSON report output by the Ace tool. This report is contained in the file named report.json (see Automated Inspection for more information about this file).

To load the report, you can drag the file from your file system and drop it in the blue box under the "Start an Evaluation" heading:

If you are not able to drag and drop the report, it is also possible to select it from the file system.

Once a file is dropped into the application, or selected from the file system, the SMART tool will automatically send the data to the server to load a new evaluation form. The SMART tool will attempt to customize evaluation interface based on the information in the report, and, when complete, an alert will be generated indicating what optimizations have been made, if any.

Note: If automatic detection results in incorrect tests being disabled, they can be re-enabled in the Conformance Verification tab.

Note: If accessibility metadata is already present in the publication, the SMART tool will not attempt to infer additional metadata.
