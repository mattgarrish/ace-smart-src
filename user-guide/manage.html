Managing Reports

There are going to be times when you are not able to complete an evaluation at the same time that you initiate it. Similarly, any time have to close down your browser during the day, any unsaved information is lost. To avoid having to leave your browser open and computer running to work around this dilemma, the reporting tool provides the ability to save and reload in-progress reports.

To save the current report, click the Save button at the right side of menu bar.

A new window will open with the JSON-encoded report data.

You can either save this page to disk directly from the browser using its "save as" option, or copy the JSON data from the text field and save it using a text editor.

Note: Do not use a word processor, like Microsoft Word, to save the data, as it may reformat the quote characters and break the JSON encoding.

Caution: Do not bookmark the HTML page, as you cannot return to it later to retrieve your data. It is a temporary page, so your report data will no longer be available when you return.

The process to reload a report is exactly the same as loading the Ace report. Either drag the saved report onto the file load location on the Start tab, or use the provided link to locate it on the file system.
