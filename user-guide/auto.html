Automated Evaluation

Before beginning the manual evaluation process, ensure that the publication passes the Ace automated evaluation tool without error.

The latest version of the tool can be downloaded from the Ace GitHub releases page.

Instructions on how to run the tool and interpret its reports is available from tbd.

To expedite manual evaluations, it is necessary to refer to the additional information provided in these reports (e.g., the list of images and alternative text). Keep the HTML report open as you proceed.

Note: Although it is possible to evaluate a publication without using the Ace automated evaluation tool, it is not recommended. The automated tool cannot check every failure, but it prevents oversights of the common errors it can detect.
