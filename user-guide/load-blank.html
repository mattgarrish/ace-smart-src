Loading a Blank Evaluation

Although it is possible to evaluate a publication without using input from the Ace evaluation tool, it is not recommended. Automated inspection ensures that any commonly-detectable errors are not accidentally missed.

To create a blank evaluation, first locate and click on the link below the drag and drop box:

You will then be prompted to provide a title for the evaluation (a title is required in order to log the evaluation in the history table):

After clicking the Ok button, a new blank evaluation will be opened.
