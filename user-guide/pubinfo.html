Adding Publication Information

The next section on the Start tab describes the publication.

The first field in this section identifies the format of the EPUB publication:

The format does not change the evaluation criteria, but it does control the output format of the optional metadata that can be generated from the tool (i.e., discovery and conformance metadata for the package document).

The rest of the section allows you to input metadata about the publication.

This metadata is used to accurately identify the publication in the output report.

Note: The addition of metadata can be skipped if you do not intend to generate a final report. To avoid validation errors, however, it is recommended to always input the required fields.

In addition to the provided fields, it is also possible to add optional metadata using the text box at the end of the form. To add optional metadata, type a human-readable label followed by a colon, a space, and then the value.

Note: As the optional metadata is for human consumption, it is not recommended to use property names (e.g., use a label like "Created On" instead of "dcterms:created").
