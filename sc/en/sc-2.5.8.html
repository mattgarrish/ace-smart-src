Every target (e.g., links, buttons) in an EPUB publication, must be at least 24 pixels by 24 pixels in size.

If a target does not meet this minimum size, the success criterion can still pass if one of the following is true:

  • The target is inline in a block of text. This exception means that links in paragraphs are exempt, for example, and do not need to be checked. (Note that lists of links would not fall under this exception.)
  • When there are adjacent targets, the target offset (the distance from one edge of the target to the next) is at least 24 pixels. The offset is measured from the furthest edge of the target to the nearest edge of the adjacent target.
  • The positioning of the targets is essential to the information being conveyed (e.g., targets on a map cannot be moved).
  • The target is a native HTML control that has not been modified by the author.
  • There is another control in the same document that provides the same functionality and meets the sizing requirement.
