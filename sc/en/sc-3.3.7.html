Check if the publication - not the reading system - requires the user to log in to perform any actions (e.g., completing an embedded quiz).

If yes, the login process cannot require the user to perform a cognitive function test (e.g., remember a password or repeat a gesture pattern) unless one of the following is true:

  • there is an alternative means of logging in that does not require such a test;
  • there is a mechanism to help the user perform the test (e.g., if a reading system allows users to store and fill in login information);
  • the cognitive function test only requires the user to recognize an object (e.g., to select an image of a car or motorcycle);
  • the cognitive function test only requires the user to recognize non-textual content that the user provided (e.g., an image or audio clip they uploaded as part of creating their account).
